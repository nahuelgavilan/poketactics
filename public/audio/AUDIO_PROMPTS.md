# PokéTactics - Audio Generation Guide

## Theme Guidelines (IMPORTANT - Use in ALL prompts)

**Estilo unificado:**
- Era GBA (Game Boy Advance) - años 2003-2006
- Chiptune moderno con elementos orquestales sutiles
- Inspiración: Pokemon Ruby/Sapphire + Fire Emblem + Advance Wars
- Energético pero estratégico, nunca agresivo
- Paleta sonora: sintetizadores retro, percusión 8-bit, melodías memorables

**Keywords consistentes para todos los prompts:**
```
GBA-style, chiptune, 16-bit, retro game, tactical RPG, Pokemon inspired,
Fire Emblem style, nostalgic, video game soundtrack, loopable
```

## Tips para generación

### Suno.ai:
1. Usa los tags sugeridos para mejor resultado
2. Genera 2-3 variantes y elige la mejor
3. Para loops, verifica que el final conecte con el inicio
4. Descarga en MP3 alta calidad

### ElevenLabs / Alternativas:
1. Si ElevenLabs no da buenos resultados, prueba:
   - freesound.org (gratis, busca "8-bit" o "retro game")
   - Pixabay Audio (gratis)
   - zapsplat.com (gratis con cuenta)
   - sfxr.me (generador de SFX retro gratuito) ⭐ RECOMENDADO para SFX
2. Normaliza el volumen de todos los SFX al mismo nivel
3. Formato: MP3 o OGG, 44.1kHz
